SQL注入

什么是SQL注入

sql注入的意思是原本的sql语句被用户的输入控制。

服务器错误地拼接用户提交的恶意参数，使sql语句原本的运行逻辑被改变，导致攻击者可以部分或者完全掌控SQL语句的执行效果。

通常服务器会根据用户提交的参数进行增删改查等操作，但如果用户通过引入闭合符，在保证原有的语法正确的情况下，引入新逻辑的语句执行，达到想要的效果。

SQL

结构性查询语言(Strucured Query Language)简称SQL。关系型数据库的标准语言，如Oracle，Mysql，Microsoft SQL Server，Access，SQLite，DB2，ngres，Sybase，PostgreSQL等。

数据库提供了一个数据检索和存储的作用。
